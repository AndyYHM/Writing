# 1. 简介
## 1.1 什么是GFM
GFM是GitHub Flavored Markdown的缩写，意味GitHub 风格的 Markdown 语法。对比标准Markdown有诸多加强和扩展；
## 1.2 什么是Markdown
Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。
